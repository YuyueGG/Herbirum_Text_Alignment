# Dataset Access

The full dataset used in this project is provided separately from the main repository.

Temporary download link:
[Google Drive dataset link](https://drive.google.com/drive/folders/1uKQd4RO2eWwxaCXMOM09sCnJzB_eXAZF?usp=sharing)

Dataset name: **Cyrtandra-44**

Description:
This dataset contains herbarium specimen images of *Cyrtandra* collected from two regions: Sulawesi, Indonesia, and Peninsular Malaysia and Singapore. The benchmark used in the paper contains 44 classes.

Notes:
- The repository only includes sample JSONL files and label mappings for reference.
- The full dataset is distributed separately due to file size and release management considerations.
- A final curated archival release may be provided later.

Included in this repository:
- sample JSONL files: `examples/sample_jsonl/`
- label mappings: `examples/sample_jsonl/label2id.json` and `examples/sample_jsonl/label2id9.json`
